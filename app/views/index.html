Service refresh assets

This resource is for developing and sharing content and front end assets for the service refresh project.

It is not a protoype for testing with users and it is not a prescriptive reference for testing.
How to use this resource

Product, BA and Research

  • Browse assets by reference

Development

  • Copy and paste markup not just text as this may contain the required attributes to ensure accessibility
  • Where new or modified CSS is required details will be stated

Content and Design

  • Create an entry in the contents table, linking to a corresponding page
  • Add a hyperlink to this page in the corresposnding JIRA ticket
  • Develop content or front end assets within the corresponding page working closely with development
  • Subsequent iterations will require changes to the user story and acceptance criteria or a new JIRA ticket number
